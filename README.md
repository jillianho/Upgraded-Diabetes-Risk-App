# 🩺 GlycoCast - Diabetes Risk Calculator

A personalized diabetes risk estimator built with Streamlit and scikit-learn.

## ✨ What's New in V2

| Feature | V1 (Pima) | V2 (NHANES) |
|---------|-----------|-------------|
| **Sample Size** | 768 | 5,000+ |
| **Genders** | Female only | ✅ Both |
| **Ethnicities** | Pima Indian only | ✅ White, Black, Hispanic, Asian, Other |
| **Data Year** | 1980s | 2009-2012 |
| **Features** | 8 | 12 |
| **Calibration** | None | ✅ Isotonic calibration |

### New Input Fields
- **Demographics**: Sex, Ethnicity
- **Medical History**: High BP, High Cholesterol, Stroke, Heart Disease
- **Lifestyle**: Smoking, Physical Activity, Diet, Sleep

### What's Preserved
- ✅ Your proxy system for estimating glucose/insulin
- ✅ FINDRISC-based logic
- ✅ Results dashboard with tabs
- ✅ "What if?" simulator
- ✅ Ranked action plans
- ✅ Value provenance tracking

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Train the V2 model (required first time)
python train_model_v2.py

# Run the app
streamlit run app.py
```

## 📁 Files

| File | Description |
|------|-------------|
| `app.py` | Main Streamlit app (upgraded with V2 features) |
| `train_model_v2.py` | Training script using NHANES data |
| `diabetes_proxies.py` | Your proxy estimation system (unchanged) |
| `model_v2.pkl` | Trained V2 model (generated by training script) |
| `model.pkl` | Original V1 model (fallback) |

## 🔧 How It Works

The app automatically detects which model is available:
- If `model_v2.pkl` exists → Uses V2 model with all new features
- If only `model.pkl` exists → Falls back to V1 model

Your `diabetes_proxies.py` is still used for:
- Estimating glucose/insulin when user doesn't know their values
- Generating the results dashboard content
- FINDRISC calculations
- Action plan generation

## 📊 V2 Model Details

**Dataset**: NHANES 2009-2012 (National Health and Nutrition Examination Survey)

**Features**:
1. `pregnancies` - Number of pregnancies (0 for males)
2. `bmi` - Body Mass Index
3. `blood_pressure` - Diastolic BP (mmHg)
4. `age` - Age in years
5. `sex` - 0=Female, 1=Male
6. `ethnicity` - 0=White, 1=Black, 2=Hispanic, 3=Asian, 4=Other
7. `high_bp` - High blood pressure diagnosis
8. `high_chol` - High cholesterol diagnosis
9. `phys_active` - Physically active
10. `smoker` - Smoking history
11. `gen_health` - General health (1=Excellent to 5=Poor)
12. `sleep_trouble` - Sleep trouble

**Algorithm**: Gradient Boosting + Isotonic Calibration

## ⚠️ Disclaimer

This tool provides estimates only based on population health data. It is not a medical diagnosis. Please consult a healthcare provider for proper diabetes screening.
